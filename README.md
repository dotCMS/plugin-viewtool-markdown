# Github Flavored Markdown plugin

## Description
This dotCMS plugin provides a github flavored markdown (gfm) compatible markdown parser (for the most part).  It is built off of the txtmark project  https://github.com/rjeschke/txtmark by Ren√© Jeschke.  It provides a viewtool that can be used in dotCMS to parse text/content for gfm.
